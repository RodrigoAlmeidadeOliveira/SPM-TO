# SPM-TO

Material de apoio para o projeto SPM-TO, incluindo planilhas, gráficos e relatorios utilizados no acompanhamento do instrumento Sensory Processing Measure (SPM) para o Tocantins.

## Estrutura

- `DOCTOS/`: planilhas, gráficos e arquivos de referência do projeto.

## Como contribuir

1. Clone o repositório.
2. Crie um branch com a sua contribuição.
3. Envie um pull request descrevendo as alterações.
